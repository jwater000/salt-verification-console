# 도서 발췌

## 1) 새로운 논리 지도
`SALT Logic System ERD`에서 정리한 본문 흐름은 `00`이 앞문을 열고, `01~05` 문제 중심, `06~11` 실마리·공리, `12~16` 통합 해법, `17` 이론 코어, `18~20` 예측·FAQ·결론, `21~28` 부록 방어의 흐름으로 이어집니다. 이 페이지는 그 흐름을 7개의 블록으로 나누어 소개하고, 대응하는 도해와 카드를 함께 볼 수 있게 정리합니다.

| 블록 | 장 구성 | 대응 도해 | 집중 포인트 |
| :--- | :--- | :--- | :--- |
| Intro | `00` | g00 | 전체 문제 의식과 자리매김, `00_결론_보고`로 연결되는 안내 |
| Part 1 – Problem Thrust | `01~05` | g00~g05 | 힘에 대한 위기 진단, 스케일/경로 감각을 매핑한 산출물 |
| Part 2 – Clue & Concept | `06~11` | g06~g11 | 관측 실마리가 공리로 전환되는 다리, 시간·양자·질량 해석 |
| Part 3 – Unified Solution | `12~16` | g12~g17 | 통합 지도(`g12`)와 모드별 흐름(g13~g16) + `g17` 고리 |
| Theory Core | `17` | g17, g18 | 양자/GR 정합과 이론적 기반, `/engineering` 실용 시나리오와 이어짐 |
| Prediction & Closure | `18~20` | g18~g26 | 검증 대기 가설, FAQ, 결론이 하나의 검증 서사로 구성 |
| Appendix & Defense | `21~28` | g27~g33 | 용어/논문/감사/예측 프로토콜 등 공식 방어 명세 |

## 2) 한눈에 보는 개념 흐름
```text
문제 제기 → 솔루션 전체 지도 → 이론 코어 → 예측/FAQ/결론 → 부록 방어
```

이 페이지는 위 흐름에 맞춰 g00~g33 이미지를 번호 순서대로 제공합니다.
- 이미지는 바로 보기
- 번호는 책 전개 순서와 맞춤
- 최신 책 본문에서 실제 참조하는 도해 번호 체계를 그대로 따름

## 4) 독자 안내
책을 처음 읽는 경우 참고할 수 있는 순서는 아래와 같습니다.
1. `00`로 컨텍스트와 결론 프레임을 잡고 `/verification/results` 페이지로 흐름을 확인
2. `01~11`으로 문제/실마리/개념 구간을 `Part 1~2` 카드와 g00~g11로 점검
3. `12~17`에서 통합 지도와 이론 코어를 보고, `g12~g18`로 각 모드의 해석을 이해
4. `18~20`을 `/verification/pending` 및 FAQ 블록으로 읽으며 예측·결론을 확인
5. `21~28`은 `/audit/*`과 `/audit/reproduce`에서 방어적 증거를 교차 확인한다
