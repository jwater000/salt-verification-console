# 재현 방법

## 1) 1분 요약
이 페이지의 목적은 하나입니다.  
`같은 데이터 + 같은 코드 + 같은 해시`로 실행해서, 같은 결론이 다시 나오는지 확인하는 것.

성공 기준:
1. `run_model_eval.py` 실행 완료
2. lock 검증 통과
3. frozen manifest 검증 통과

## 2) 바로 실행 (필수 3단계)
### 1단계: 평가 실행
```bash
.venv/bin/python tools/evaluation/run_model_eval.py
```

### 2단계: prediction lock 검증
```bash
.venv/bin/python tools/evaluation/verify_prediction_lock.py
```

### 3단계: frozen manifest 검증
```bash
.venv/bin/python tools/evaluation/verify_frozen_manifest.py --manifest data/frozen/current/manifest.json
```

## 3) 결과가 같다고 판정하는 방법
아래가 모두 맞으면 같은 결론으로 봅니다.
- 통합 집계: `56 / 14 / 12` (SALT / 표준 / 동률, frozen-20260308 기준)
- 잠금 해시 4종 일치:
  - `micro_prediction_lock_sha256`
  - `micro_sm_prediction_sha256`
  - `micro_salt_prediction_sha256`
  - `frozen_manifest_sha256`

## 4) 모델식 한눈표 (왜 다른 식을 쓰는가 포함)
```text
[거시]
항목            | 식(요약)                                  | 왜 이렇게 두는가
----------------|-------------------------------------------|---------------------------------------------
표준측          | 0.78 + 0.09z + 0.015*d/(d+1500)           | 완만한 선형+포화 기준선
SALT            | 0.77 + 0.10z + 0.02*d/(d+1200) - 0.004z^2 | 고 z 구간 곡률 잔차 가설 검정
표준 fallback   | 0.81 + 0.18u - 0.015(1-u)^2               | 결측 시 보수적 다항 근사
SALT fallback   | 0.79 + 0.20u^1.08 - 0.008u(1-u)           | 결측 구간 민감도의 약한 비선형 검정

[미시]
항목            | 식(요약)                                  | 왜 이렇게 두는가
----------------|-------------------------------------------|---------------------------------------------
SM muon g-2     | (QED+EW+HVP+HLbL)*1e-11                   | 문헌 기반 표준 분해합 기준선
SALT muon g-2   | (SM components + delta)*1e-11             | 추가 유효기여(delta) 가설 검정
SM tail         | (450/x)^1.15                              | 보수적 감쇠 지수 기준
SALT tail       | (450/x)^1.12                              | 고에너지 tail 잔차 개선 가능성 검정
```

## 5) 표준식 출처 / SALT 가정 근거
표준식 출처:
- muon g-2 분해합 문헌:  
  `Aoyama et al., Phys. Rept. 887 (2020) 1-166, doi:10.1016/j.physrep.2020.07.006`
- 관측 출처 묶음: PDG, NuFIT, T2K, NOvA, IceCube, Super-K, Daya Bay, Double Chooz, RENO
- 거시 관측 출처 묶음: GraceDB, GCN, HEASARC

SALT 가정 근거:
- 표준 입력은 그대로 두고, 잔차 패턴(특히 고 z/고에너지 구간)이 체계적으로 줄어드는지 보려는 최소 추가항(곡률/지수/delta)만 넣음
- 즉 “새 이론을 선언”하기보다 “같은 데이터에서 대안 함수형이 더 잘 맞는지”를 계량 비교
