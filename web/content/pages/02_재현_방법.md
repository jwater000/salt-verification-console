# 재현 방법

## 1) 1분 요약
이 페이지는  
`같은 데이터 + 같은 코드 + 같은 해시`로 실행했을 때 같은 산출 경로가 나오는지 확인하는 절차를 정리합니다.

성공 기준:
1. `run_model_eval.py` 실행 완료
2. lock 검증 통과
3. frozen manifest 검증 통과
4. 같은 dataset/formula/engine 잠금값이 유지될 것

## 2) 바로 실행 (필수 3단계)
### 1단계: 평가 실행
```bash
.venv/bin/python tools/evaluation/run_model_eval.py
```

### 2단계: prediction lock 검증
```bash
.venv/bin/python tools/evaluation/verify_prediction_lock.py
```

### 3단계: frozen manifest 검증
```bash
.venv/bin/python tools/evaluation/verify_frozen_manifest.py --manifest data/frozen/current/manifest.json
```

## 3) 결과가 같다고 판정하는 방법
아래가 모두 맞으면 같은 결론으로 봅니다.
- 통합 집계: `56 / 14 / 12` (SALT / 기준 이론 / 동률, frozen-20260308 기준)
- 잠금 해시 4종 일치:
  - `micro_prediction_lock_sha256`
  - `micro_sm_prediction_sha256`
  - `micro_salt_prediction_sha256`
  - `frozen_manifest_sha256`

추가 확인:
- `engine_version`
- `formula_version`
- `dataset_version`

## 4) 재현 프레임 한눈표
```text
[거시]
항목            | 재현 기준                                  | 왜 이렇게 두는가
----------------|-------------------------------------------|---------------------------------------------
기준 비교축     | Shapiro/적색편이/렌즈/GNSS/GW-EM          | 같은 관측 채널에서 기준식과 비교
SALT 비교축     | c_eff(rho) 기반 밀도-전달률 해석          | 단일 프레임으로 잔차 구조 비교
판정            | 같은 관측값에 대한 잔차/적합도 비교        | 같은 데이터에서 함수형만 비교

[미시]
항목            | 재현 기준                                  | 왜 이렇게 두는가
----------------|-------------------------------------------|---------------------------------------------
기준 이론       | SM + 공개 문헌/실험 기준값                | 같은 관측량에서 기준선 확보
SALT 비교식     | 보정항을 포함한 비교식                     | 난제 설명력과 적합도 개선 여부 확인
판정            | chi^2 / RMSE / FDR(BH)                    | 통계 정합과 해석 우위를 함께 판정
```

운영 잠금 메모:
- 책의 27장은 실제 차별 예측을 3개 채널만 운영 잠금합니다.
- 웹의 공개 결과는 그중 현재 데이터/코드/해시가 준비된 채널만 집계합니다.
- 설명용 가설은 `/verification/pending`으로 분리합니다.

## 5) 표준식 출처 / SALT 가정 근거
표준식 출처:
- muon g-2 분해합 문헌:  
  `Aoyama et al., Phys. Rept. 887 (2020) 1-166, doi:10.1016/j.physrep.2020.07.006`
- 관측 출처 묶음: PDG, NuFIT, T2K, NOvA, IceCube, Super-K, Daya Bay, Double Chooz, RENO
- 거시 관측 출처 묶음: GraceDB, GCN, HEASARC

SALT 가정 근거:
- 거시는 단일 `alpha` 일관성 아래 Shapiro 지연, 적색편이, 렌즈 지연 등을 함께 비교하는 프레임으로 본다.
- 미시는 보른 규칙 수준 통계 정합을 1차 기준으로 두고, 해석 우위와 선택 채널 차별 예측을 추가로 본다.
- 즉 “새 이론을 선언”하기보다 “같은 데이터에서 대안 함수형이 더 잘 맞는지”를 계량 비교합니다.
