# Method

이 페이지는 현재 웹 콘솔이 실제로 채택한 `검증 규격`만 요약한다. 책 전체의 모든 주장이나 해석을 그대로 옮기는 곳이 아니라, 지금 웹에서 판정 가능한 항목이 어떤 규칙으로 계산되는지 보여주는 운영 문서다.

## 기준이 되는 장

- `19장`: SALT가 왜 저에너지에서 GR 계열 기술과 정합된다고 보는지의 개요
- `25장`: 상태 변수, 약장 식, 시간지연/적색편이/렌즈 해석의 최소 수학 골격
- `26장`: ADM 제약과 자유도 2개 유지에 대한 감사 시트
- `27장`: 실제 차별 예측 채널과 반증 기준을 어디까지 잠그는지에 대한 운영 프로토콜

## 웹에서 지금 고정한 운영 원칙

1. 같은 관측값에 대해 `standard_fit`과 `salt_fit`을 따로 계산한다.
2. 입력 payload는 예측값을 직접 담지 않고, 독립 예측 엔진이 값을 산출한다.
3. 공개 결과는 frozen dataset, prediction lock, manifest hash가 함께 맞을 때만 유효하다.
4. 책의 설명 문장과 별개로, 웹에서는 `실제 score로 채점 가능한 항목`만 결과 페이지에 올린다.

## 현재 웹의 판정 구조

| 축 | 기준 모델 | SALT 쪽 비교식 | 현재 웹 상태 |
| :--- | :--- | :--- | :--- |
| 거시 | GR/ΛCDM 기반 비교식 | `c_eff(rho)` 기반 지연/적색편이 해석 | 부분 운영 중 |
| 미시 | SM + 문헌 기반 baseline | 보정항을 포함한 SALT 비교식 | 운영 중 |
| 통계 | 절대오차, chi^2, RMSE, AIC/BIC, FDR | 동일 규칙 적용 | 운영 중 |
| 감사 | manifest hash, prediction lock, engine/formula version | 동일 기준 | 운영 중 |

## 19장, 25장, 26장, 27장이 웹에서 각각 맡는 역할

- `19장`은 왜 "새 방정식 완성"보다 "저에너지 정합 경로"를 말하는지의 설명 기준이다.
- `25장`은 `rho`, `n = rho^2`, `theta`, `mu`, `g_eff ∝ -nabla mu` 같은 핵심 변수와 약장 식의 기술적 배경이다.
- `26장`은 ADM 분해에서 추가 자유도와 BD ghost를 어떻게 감사하는지의 별도 근거다.
- `27장`은 초고에너지 LIV, 강중력장 추가 지연/렌즈, 초고주파 중력파 꼬리처럼 실제 반증 채널을 어디까지 잠글지 정하는 최종 운영 기준이다.

## 웹이 일부러 하지 않는 것

- 책의 해석 문장을 곧바로 `검증됨`으로 승격하지 않는다.
- 아직 관측량 매핑과 score가 없는 채널은 `/evaluation`에 올리지 않는다.
- 설명용 예측과 운영 잠금된 차별 예측을 섞지 않는다.

즉 이 페이지의 핵심은 "책의 모든 내용을 요약"이 아니라, `현재 웹이 어떤 규칙으로 계산하고 어떤 장을 그 근거로 삼는가`를 분리해서 보여주는 것이다.
