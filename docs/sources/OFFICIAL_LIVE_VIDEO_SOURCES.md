# 공식 실시간 영상 소스 목록 (LIGO/Fermilab 포함)

최종 정리: `2026-03-06`

## A. 상시/준상시 라이브 제공처
1. NASA Live (NASA+ / ISS Live 포함)  
   - https://www.nasa.gov/live
2. ESO Live Webcams (Paranal, ELT, ALMA 등)  
   - https://www.eso.org/public/outreach/webcams/
3. ALMA Live from Chajnantor (공식)  
   - https://www.almaobservatory.org/en/live-from-chajnantor/
4. NRAO VLA Webcam  
   - https://public.nrao.edu/vla-webcam/
5. NRAO ALMA Webcam  
   - https://public.nrao.edu/alma-webcam/
6. NRAO ngVLA Webcam  
   - https://public.nrao.edu/ngvla-webcam/
7. USGS Volcano Webcams (지질/화산 실시간 관측)  
   - https://www.usgs.gov/volcanoes/kilauea/webcams

## B. 이벤트성 공식 라이브 제공처
8. ESA Web TV (행사/브리핑/발사 일정 기반)  
   - https://watch.esa.int/

## C. 프로젝트 기존 소스 (요청 반영)
9. LIGO 공식 채널  
   - https://www.youtube.com/channel/UC4oFlSYpDywInX0lxpiBPwA
   - 비고: 상시 라이브가 아니라 라이브 시점에만 live stream 활성화되는 경우가 많음
10. Fermilab 공식 채널  
   - https://www.youtube.com/user/fermilab
   - 비고: 상시 라이브가 아니라 이벤트/방송 시점 중심
11. YouTube 라이브 테스트 링크 (사용자 제공)
   - https://www.youtube.com/watch?v=0FBiyFpV__g
   - 비고: 외부 사이트 임베드가 차단될 수 있어 사이트 내 iframe 재생은 실패할 수 있음

## 운영 메모
- 영상은 설명/신뢰 보강용으로 사용
- 모델 판정은 반드시 수치 데이터 API(`GWOSC`, `GraceDB`, `GCN`, `ZTF(IRSA)`, `HEASARC`) 기반으로 수행
- 라이브가 없을 때는 최근 공식 녹화영상으로 fallback
