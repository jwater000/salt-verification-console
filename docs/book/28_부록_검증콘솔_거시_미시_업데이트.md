# 28. 부록: 검증콘솔(거시/미시) 업데이트 노트

- 1단계: 거시(우주론) 검증 규격 고정

## 1) 거시(우주론) 업데이트: 웹 반영 완료 항목

### 1.1 표준이론 범위 고정
- 현재 웹 콘솔의 `Standard`는 **표준우주론(ΛCDM)**을 의미한다.
- 미시(입자물리) Standard Model(SM) 검증은 별도 트랙으로 분리한다.

### 1.2 데이터 비교 규격 (운영식)
거시 이벤트 단위 비교는 아래 3중 구조를 사용한다.

- `actual_value`: 실측값
- `standard_fit`: 표준우주론(ΛCDM) 예측값
- `salt_fit`: SALT 예측값

오차/판정은 아래 규칙으로 고정한다.

\[
\texttt{standard\_error}=actual\_value-standard\_fit
\]
\[
\texttt{salt\_error}=actual\_value-salt\_fit
\]
\[
\texttt{winner}=\arg\min\left(\left|standard\_error\right|,\left|salt\_error\right|\right)
\]

집계 지표:
- `MAE`, `RMSE`, `avg_delta_fit`

통계 기준:
- `alpha=0.05`
- `FDR(BH) q=0.10`
- 효과크기 기준 병행

### 1.3 관측 채널 정합
거시 예측식/판정 채널은 24장 13.4를 기준으로 유지한다.
- Shapiro 지연
- 중력 적색편이
- 강중력 렌즈 지연
- GNSS/GPS
- GW-EM 도착시각

참조:
- 24장 `13.4 검증 프로토콜 표`
- 26장 `예측 6.1 (LIV)`

### 1.4 웹 페이지 대응
- `Evidence`: 실측 vs ΛCDM vs SALT 비교
- `Events`: 이벤트 단위 원자료/오차/판정
- `Limits`: ΛCDM 우세/동률/한계 공개

## 2) 미시(입자물리) 업데이트: 초안 고정

### 2.1 현재 상태 진단
- 책에는 미시 채널(뮤온 g-2, 중성미자, 충돌기) **검증 방향**은 제시되어 있다.
- 미시는 보셀 해상도 한계 때문에 **관측량별 완성 예측식의 완전잠금**을 목표로 두지 않는다.
- 대신 관측량/비교식 형태/파라미터 범위/통계판정 규칙을 고정하는 **운영잠금(operational lock)** 으로 진행한다.
- 따라서 미시는 `검증대기` 트랙으로 분리 운영한다.

### 2.2 이미 보유한 미시 관련 수식/채널
- 고에너지 LIV 분산식(26장):
\[
\omega^2 = k^2 + \eta\left(k^4/M^2\right)
\]
- 미시 후보 채널(17장):
  - 뮤온 g-2
  - 중성미자
  - 충돌기

### 2.3 미시 검증 데이터 계약(v0)
미시 데이터는 아래 최소 스키마를 사용한다.

- `observable_id` (예: `muon_g_minus_2`, `nu_osc_theta23`)
- `dataset_id` (예: 실험/런 구분)
- `measured_value`
- `measurement_uncertainty`
- `sm_pred`
- `salt_pred`
- `sm_residual = measured_value - sm_pred`
- `salt_residual = measured_value - salt_pred`
- `winner = argmin(|sm_residual|, |salt_residual|)`
- `quality_flag`
- `source_url`
- `observed_at_utc`, `ingested_at_utc`

### 2.4 미시 반증 규칙(v0)
- 같은 관측량에서 `|salt_residual| < |sm_residual|` 우세 비율을 채널별로 집계한다.
- 채널별 `MAE/RMSE`를 계산하고 다중비교 보정(FDR) 후 판정한다.
- 특정 채널에서 SALT 우세가 반복 재현되지 않으면 해당 채널 가설을 보류한다.

### 2.5 미시 다음 작업(운영잠금 필요)
아래 3개를 잠그면 웹 미시 검증을 시작할 수 있다.

1. 관측량/비교식 형태/파라미터 범위(뮤온 g-2, 중성미자 진동, 충돌기 잔차)
2. 기준 SM 예측 소스(HEPData/PDG/Fermilab/CERN Open Data 등)
3. 채널별 반증 임계값(통계량/유의수준/효과크기)
