# Paper Submission Readiness Checklist (Physics Review)

최종 업데이트: `2026-03-09`

목적:
- 미시/거시 이론물리학자 리뷰에서 즉시 지적될 위험을 제출 전에 제거한다.

## A. 예측식 타당성
- [ ] 채널별로 표준식 1차 출처(DOI/협업 공식문서) 명시
- [ ] 채널별로 SALT식 유도 단계 문서화
- [ ] 표준식/SALT식이 같은 관측량 정의를 공유
- [ ] 차원 분석(단위 일치) 완료
- [ ] 극한 거동(저/고에너지, 경계조건) 점검 완료
- [ ] 식의 적용 범위와 실패 조건 명시

## B. 통계/판정 타당성
- [ ] 사전고정된 판정 규약(alpha/FDR/effect size/tie) 사용
- [ ] 다중비교 보정(FDR) 반영
- [ ] 효과크기와 유의성 동시 보고
- [ ] 부정 결과(표준 우세/동률/insufficient_data) 누락 없이 공개

## C. 재현성/투명성
- [ ] `engine_version`, `formula_version` 고정
- [ ] `prediction_lock_sha256`와 frozen manifest 해시 공개
- [ ] 재실행 커맨드 1회 실행으로 동일 산출물 생성 가능
- [ ] 독립 재현(내부팀 외) 결과 1건 이상 확보
- [ ] 데이터 소스 URL/버전/라이선스 공개

## D. 리뷰어 예상 비판과 대응 증거
- [ ] “표준식 구현이 틀렸다” 비판 대응: 기준 문헌 식-코드 매핑표
- [ ] “SALT식이 임의 보정이다” 비판 대응: 유도 단계/자유파라미터 정책
- [ ] “사후튜닝했다” 비판 대응: prereg + amendment 로그
- [ ] “선택적 보고다” 비판 대응: 전체 채널 결과표와 실패 공개
- [ ] “재현 불가” 비판 대응: lock hash + 명령 + 환경정보

## E. 제출 직전 Go/No-Go
- [ ] 핵심 채널(거시/미시) 모두에서 식 문서 완성
- [ ] 코드-문서 불일치 0건
- [ ] 통계 규약 위반 0건
- [ ] 재현 검증 실패 0건
- [ ] 공동저자 내부 리뷰 승인

No-Go 규칙:
- 위 항목 중 A/B/C에서 하나라도 미충족이면 제출 보류
